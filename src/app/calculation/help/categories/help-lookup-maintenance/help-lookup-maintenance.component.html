Overview

The lookup tables are stored against the user account. On creation of a new one the user can access a blank table which can create the relevant columns and rows which can be looked up as part of the calculation. The user can add in values of a table by importing a CSV file.



Add Column

The first thing to do is to add a column, so click the add column button on the top row which will display the following input box to the user:



Add in a name for the column, this will be selectable from the configuration so ensure that a name is used to identify the column you need.

Add Row

Once a column has been added in a new row is required, add in a new row of data required for the lookup.

Delete All

Selecting this will remove all values within the table.

Overview

The Lookup Maintenance table has the ability to import the lookup values from a .CSV file or export the table contents to a .CSV file.

Export CSV

The lookup table can be exported to a CSV file, this function is used so that a lookup table can be amended in a CSV file before being imported back into the Lookup Table. Click the button and the .csv file will go into the download queue from you web browser.



Import CSV

The user can import the lookup table from a CSV file by clicking the Choose File button and selecting a CSV file. This will use the column headings from the first row of the file as the column headings in the lookup table.
