Overview

The factor tables are stored on the server as .CSV files, these files can be used directly from a scheme’s factor tables by converting them to a .CSV file. To access the factor maintenance screen select Maintenance on the header menu.

Download Table

The Download Table section provides a list of factor tables available in the application; this enables the user to view the factor tables on the server. This can be used to update these factors in the future or view the tables to configure the factors function in the configuration builder. Select the table you want to view, and click download to download the .CSV to the browser.

Upload Table

The Upload Table section allows the user to select a .CSV file from the user’s desktop to upload the .CSV to the application. If a table exists currently this will be overwritten.

Delete Table

The Delete Table section allows the user to delete a .CSV file from the application.
