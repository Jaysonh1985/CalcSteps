Releasing Configuration

Each time the configuration is saved it creates a new sub version, when we are looking to release this to the run time environment, select the Release button on the Configuration menu. This will then upgrade the version to a full version and release this to the new environment.

Running Calculations

This will then be available in the Release Environment; this menu also allows the user to search for particular released calculations and sorting on the Scheme, Name and Updated columns. To access this click on the row to view. The main page on the release shows as below, Calculation Bulk which will be covered in a later section. Calculation Input shows the dynamic input form from the configuration builder and allows the user to Input the data for the calculations and Calculation Output will show the results of the calculation. Calculation Input The Calculation Input shows an input form and has the same validations as in the builder so that Dates and Decimals are validated to make sure that they are correct before submitting the calculations.

Running Bulk Calculations

All calculations can be run in bulk, using the Calculation Bulk section of the calculation. The user can click the export template; this will provide a .csv file where a user can input the data for multiple members. Once the file has been created and member information has been added the user should then select the file to Import by clicking the Choose Files button. Once this has been found the click the download to perform the calculation and produce the output csv file.
